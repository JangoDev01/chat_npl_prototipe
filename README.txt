para entregar possiveis tratamentos ou algum outro tipo de dados relacionado a saude do usuario...

=== Funcionalidades do chat ===

1 - O chat começa com uma mensagem padrão do bot perguntando algo com:
    "Diga-me o que sentes para eu poder ajudar..."

2 - O usuario digita os sintomas e aguarda...

    2.1 - Caso o texto digitado seguir as normas éticas estabelecidas no chat...
        - Os dados consernentes aos sintomas serão entregues ao usuario...

    2.2 - Caso o texto digitado não seguir as normas éticas estabelecidas no chat...
        Ex: Ofença, insultos, ataques de odio ao chat ou a terceiros, etc...
        - O chat bloqueia com uma mensagem de alerta...

    2.3 - Caso o texto digitado não ter nenhuma relação com a saude, fisica ou menta...
        - O chat envia uma mensagem aconselhando o usuario a não sair do assunto...
